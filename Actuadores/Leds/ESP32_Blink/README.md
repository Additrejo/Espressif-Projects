## Descripción y objetivos

Este proyecto contiene el firmware para encender un led.

* **Objetivo Principal:** Encender un led con ESP32.
* **Objetivos Secundarios:**
  * Modificar la velocidad del blink.
  * Conectar más leds y controlar sus velocidades.

## 🛠️ Stack Tecnológico y Componentes

### Software
* **Firmware (IDE):** [VS Code + PlatformIO](https://code.visualstudio.com/) | [Arduino IDE](https://docs.arduino.cc/software/ide/) | [ ESP-IDF](https://idf.espressif.com/)
* **Esquematicos:** [Cirkit Designer IDE](https://app.cirkitdesigner.com/)
* **Librerias:** No se necesita una libreria.


### Hardware y Componentes Clave
* **Microcontrolador:** ESP32 (30 PIN) 
* **Actuadores:** Leds (Diferentes colores)
* **Componentes:**  Resistencias 220Ohms

## 💡 Uso y Operación

Realiza paso a paso lo siguiente:

1. Construir el siguiente circuito.
   ![alt text](<img/ESP32 Blink Esquematico.png>)

2. Conecta tu computadora a la PC.

3. Abre el IDE de tu preferencia.

4. Compila y sube el código: **[ESP32_Blink](https://github.com/Additrejo/Espressif-Projects/blob/main/Actuadores/Leds/ESP32_Blink/src/ESP32_Blink.cpp)**  a tu IDE

5. Observa los resultados.

Si funciona bien, felicidades, no olvides etiquetarnos en tu proyecto.
